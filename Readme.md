### 温湿度のグラフを作成シェルについて

Usage:
  graph-make.sh <measurementDataFile> <GraphDate>

Description:
  温湿度のデータから1日分のグラフ画像を作成するツール

measurementDataFile:
  温湿度データのファイルを指定してください。

GraphDate:
  グラフを作成したい日付を指定して下しあ。
  format: YYYY/MM/DD
